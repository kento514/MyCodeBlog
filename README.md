## コードスパン
`puts 'Qiita'` と書くことでインライン表示することも可能です。

`` `バッククオート` `` や ``` ``2連続バッククオート`` ``` も記述できます。

## 強調，イタリック，打消し戦
_ か * で囲むとHTMLのemタグになります。Qiitaでは *italic type* になります。
__ か ** で囲むとHTMLのstrongタグになります。Qiitaでは **太字** になります。  
打ち消し線を使うには ~~ で囲みます。 ~~打ち消し~~
